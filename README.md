# GamerMood 2.0 TFG

Proyecto de TFG DAM orientado al desarrollo de una aplicación web full stack para registrar sesiones de juego, analizar el estado emocional del usuario y generar recomendaciones personalizadas.

## Tecnologías previstas
- Frontend: Angular
- Backend: Spring Boot
- Base de datos: PostgreSQL
- IA: OpenAI API

## Estructura del repositorio
- `frontend/` aplicación Angular
- `backend/` API REST Spring Boot
- `database/` scripts y modelo de datos
- `docs/` documentación técnica y organizativa

## Equipo
- Integrante 1: Backend y seguridad
- Integrante 2: Base de datos y lógica de dominio
- Integrante 3: Frontend e integración

## Flujo de trabajo
- `main`: versión estable
- `develop`: integración
- `feature/*`: desarrollo por funcionalidad
